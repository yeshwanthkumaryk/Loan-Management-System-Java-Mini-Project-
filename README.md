# Loan-Management-System-Java-Mini-Project

Description

This is a simple Java-based Loan Management System developed using Core Java concepts.
The application simulates the process of checking loan eligibility for different types of loans such as:
- Personal Loan
- Home Loan
- Car Loan
It collects user details like salary, age, CIBIL score, and document verification (Aadhar, Phone, PAN) and determines whether the loan is approved or rejected based on predefined conditions.

Features

✅ Input validation using Regular Expressions (Regex)
✅ Loan eligibility check based on:
Age
Salary
CIBIL Score
✅ Multiple loan types with different criteria
✅ Object-Oriented Programming concepts:
Inheritance
Method Reusability
✅ Console-based user interaction
